## 新增条目

新增一个 artifact / 网页时：

1. 在仓库根目录新建子文件夹，例如 `new-topic/`
2. 把 HTML 文件命名为 `index.html` 放进去
3. 在根目录 `index.html` 中新增一个 `.link-card` 条目
4. 推送即可，GitHub Pages 会自动重新部署（通常 30 秒到几分钟生效）

访问链接形如 `https://yishisanren.github.io/office_links/new-topic/`。
